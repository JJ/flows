## Description
**Selections** on flow matrices, **statistics** on selected flows, **map** and **graph** visualisations.


## Installation
### From CRAN
Stable version
```{r}
install.packages("flows")
```
### From GitHub
Development version
```{r}
require(remotes)
remotes::install_github("riatelab/flows")
```
